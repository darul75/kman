KMAN

KMAN Stack (Koa, MongoDB, Angular.js, Node.js): Full stack JavaScript Web development boilerplate.

Koa framework provides the RESTful backend API, utilizing the harmony generators.
MongoDB is our NoSQL database of choice in this stack and sample app.
AngularJS is our fully static client side app framework, and it is heroic!
Finally, Node.js is super glue, runtime for the backend, built system for the front.
